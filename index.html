JavaScript Basics


Introduction

  • JavaScript is the programming language for the web
  • JavaScript can update and change both HTML and CSS
  • JavaScript can calculate, manipulate and validate data

During the formative years of the world wide web, web pages could only be static and lacked any dynamic capabilities. In 1995, the browser Netscape introduced JavaScript as a solution to this problem.

These days, JavaScript is the dominant client-side scripting language of the web with 95% of websites using it.

What it does

The following are some examples of what JavaScript can do;

  • Load new page content without refreshing the page
  • Animate page contents such as fading, resizing, moving, etc.
  • Add interactive content like games or videos
  • Transmit information regarding the user's behaviour for ad tracking, personalization and analytics
  • Validate inputs from a web form to ensure correctness before submitting to the server

Where to

In HTML, JavaScript code can be written directly into HTML generally before the closing body tag using a style tag;

eg. <style> enter your JavaScript code here </style>

The preferred method is to link a separate JavaScript page using a script tag to keep your HTML page clean.

eg. <script src="scripts/main.js"></script>

Either of these options are generally entereed before the closing body tag to ensure all HTML is loaded prior to running any JavaScript.

Data Types

In JavaScript, the concept of data types is an important one. To be able to operate on variables and create strings successfully, we must be able to understand the type of data we need. The following four data types are considered Primitive Data;

Strings

A string (or a text string) is a series of characters like "John Doe". Strings are written with single or double quotes.

Numbers

A number is represented without quotes and contain only number values. They can be written with or without decimal points.

Booleans

Boolean values can only represent two things; true or false.

Undefined

In JavaScript, a variable without a value has the value and type of undefined. Any variable can be emptied using the value and type undefined

Comments

JavaScript comments can be used to explain code and make things more readable. Comments can also be used to prevent execution when testing alternate code.

Single Line Comments

Single line comments start with //. Anything entered between // and the end of the line will not be executed.

eg. //anything after this will be greyed out

Multi-line Comments

All multi-line comments start with /* and end with */. Any text placed between the two will be ignored.

eg. /*All of this text is purely comment only and can span multiple lines of code */
